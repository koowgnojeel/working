To flip a video upside down (vertical flip) in Linux using FFmpeg

* Vertical Flip (Upside Down)
  ffmpeg -i input.mp4 -vf "vflip" output.mp4  # This flips the video up-and-down.

* Horizontal Flip (Mirror)
  ffmpeg -i input.mp4 -vf "hflip" output.mp4  # This flips the video left-to-right.

* Rotate 180 Degrees (Upside Down)
  ffmpeg -i input.mp4 -vf "transpose=2,transpose=2" output.mp4  # using the transpose filter twice achieves a 180-degree turn. 

Key Considerations - Re-encoding
    Using -vf (video filter) will re-encode the video,
    which takes time
    but ensures the flip is applied permanently.
